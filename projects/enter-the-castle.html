Enter The Castle

Project Description

This solo project is currently developped by myself. The goal is to do an immersive sim, like Dishonored or Deus Ex, one of my favourite games ! For now there is only a test map with few features. But the best is yet to come ! The Game is hosted on Itch.io with a web player. Try it !

Main features

  • Improving behavior of cars and pedestrian IA
  • Create tools in Unity Editor to configure and save different scenarios
  • Creation of pipeline importation in order to add new model of cars

Next main tasks

  • Use of Unity
  • Use of Unity Editor to create edition tools
