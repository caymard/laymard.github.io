Enter The Castle

Project Description

This solo project is currently in development. The goal is to do an immersive sim, like Dishonored or Deus Ex, one of my favourite games ! For now there is only a test map with few features. But the best is yet to come ! The Game is hosted on Itch.io with a web player. Try it !

Main features

  • FPS controlled based on Unity Standard Assets
  • Tools and components to create interaction and trigger based on condition
  • Melee attack and dash teleport for the player

Next main tasks

  • Enemy IA (attack and detection of player)
  • Range attack ( throw the axe like in God Of War)
  • Creation of small levels
