Dead By Daylight Mobile

Project description

We are currently making the adaptation of Dead By Daylight into mobile platforms. We have to recreate the same gameplay experience with all constraints mobiles impose us.

Contributions

  • Making the HUD with the touch interface
  • Adapted friends and party systems from PC and console versions to mobile platforms. Integrating corresponding user interface in collaboration with UI artists
  • Create a process to use texture atlases in order to improve UI memory usage
  • Involved in the code design for sharing code between mobile and PC/Console
  • Helped on the tutorial level scripting
  • Profiled CPU and network performance

Development details

  • Code in C++
  • Usage of Unreal Engine, TexturePacker, FramePro, Android Studio
  • Perforce for version control
